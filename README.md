#bingBackgroundImage

获取bing首页每日一图

----------------------------------------------------------------------------------------

名称：bingBg1920

大小：1920*1080

获取接口：http://cn.bing.com/HPImageArchive.aspx?format=js&idx=0&n=1  返回json数据

----------------------------------------------------------------------------------------

名称：bingBg1366

大小：1366*768

获取接口：http://cn.bing.com/HPImageArchive.aspx?idx=0&n=1  返回xml数据

-----------------------------------------------------------------------------------------

idx值可选-1 0 1，分别对应 明天，今天，昨天。
